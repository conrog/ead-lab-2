Lab 3 - AJAX and JSON

This website is served from a Node.js server running on port 8080.

The website offers several functionalities which are implemented using technologies such as AJAX, jQuery and JSON. jQuery is used to perform AJAX requests to the server for the json files which are contained within the country-objects directory. These requests are handled by the server via a request handler, which returns the requested document (html, js, css and json).

jQuery is used for a number of functions on the website. It is used to

  • Make AJAX requests to the server.
  • Change the websites colour theme.
  • Fade in the counties table when loaded.
  • Show 20 or all table rows when button is clicked.
  • Fade out the loading status message when clicked.
  • Change the table cell colours to the selected colour when clicked and revert when clicked again.
  • Change disabled property of "Show More" button.
  • Show/hide the summary of each of the technologies used sections.

Below you can find a brief summary of the technologies mentioned above in this project. Click on the headings to display the content.

AJAX ▼

AJAX stands for Asynchronous JavaScript And XML. It is used in the browser to send data to a web server in the background. This enables websites to update sections of the page without having to reload the whole page.

AJAX makes use of the XMLHttpRequest object which is built into all browsers to request data from a web server. When the server returns a response, JavaScript is then used to process the data and update the webpage.

JSON ▼

JSON stands for JavaScript Object Notation. It is a lightweight text based format used for storing and transmitting data. It was derived from as the name suggests JavaScript object notation, however, the format differs slightly, it is text based which can be seen in the examples below

JavaScript Object:

{ name: "Conor", age : 25, address: "123 Fake Street"}

JSON Object:

{ "name": "Conor", "age" : 25, "address": "123 Fake Street"}

jQuery ▼

jQuery is a library which aims to make it easier to use JavaScript on a website. It is lightweight and was developed with the idea of "write less, do more".

jQuery takes a lot of the common tasks which can take serveral lines to write in vanilla JavaScript, and provides the same functionality through its various methods. These common tasks include the likes of DOM manipulation, CSS manipulation, Events (button clicks, mouse over, etc..), animations and AJAX requests.

Node.js ▼

Node.js is a free, open-source server environment which enables developers to write their server applications using JavaScript. Node.js is runs a single-thread, non-blocking, and offer asynchronous programming and as a result is very memory efficient.

Node.js can generate webpages dynamically, perform file operations (create, read, write, delete and close) on the server, collect form data, and can also perform CRUD database operations.
